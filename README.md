# Remote Traffic Lights

Демонстрационный пешеходный переход на ESP32 и ESP-NOW.

## Архитектура

Два одинаковых переносных столба. На каждом:
- Транспортный светофор (R, Y, G)
- Пешеходный светофор (R, G)
- Кнопка вызова для пешеходов

## Логика работы

В обычном режиме транспортный горит зелёным, пешеходный — красным.
При нажатии на любую из кнопок:
1. Транспортный начинает мигать зелёным (4 сек)
2. Переключается на жёлтый (3 сек)
3. Переключается на красный
4. Пешеходный становится зелёным (10 сек)
5. Пешеходный мигает (5 сек)
6. Возврат в исходное состояние

Столбы синхронизируются через ESP-NOW. Если связь пропадает — переходят в аварийный режим (мигающий жёлтый).

## Подключение

| Пин | Назначение |
|-----|------------|
| 4   | Кнопка (GND при нажатии) |
| 16  | Транспортный — красный |
| 17  | Транспортный — жёлтый |
| 18  | Транспортный — зелёный |
| 19  | Пешеходный — красный |
| 21  | Пешеходный — зелёный |

## Настройка адреса

После прошивки первого столба открой Serial Monitor — он покажет MAC-адрес.
Пропиши его в `peerAddress` второго столба и прошей.
